## Environment Variables

Add the following environment variables to your `.env` file:

```
# Server
PORT=
NODE_ENV=
MONGO_URI=

# JWT secret
JWT_SECRET=

# Cloudinary credentials
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=

# Google OAuth credentials
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GOOGLE_REDIRECT_URI=
```
